# v1.2.9.0

## ⚠️ 升级须知

**本次更新改动较大，升级前请先删除旧版的 `images/` 和 `cache/` 文件夹！**

程序首次启动时会自动释放最新的模板图片和重建缓存。如果不删除旧文件夹，可能残留过期模板导致匹配异常。
